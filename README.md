# MagicCardBattle
Un jeu de bataille de cartes, basé sur pygame.
Inspiré par hearthstone, magic the gathering


Les + :
Une intelligence artificielle correcte, quoique un peu lente.
La possibilité de créer un nombre illimité de cartes différentes.

Le jeu nécessite l'installation de : 
python 2.7x  (32 bits)
pil (python image library)
pygame

Pour l execution, lancer :
(python) PortailAccueil.py

Au premier lancement, le jeu recrée les cartes.

Regles du jeu dans le fichier ReglesMagicCardBatlle.odt

J'attends vos meilleurs decks pour des tournois joués par l'IA.
